⚡ Features

Everything you need.
Nothing you don't.

All the power of premium API clients, without the price tag or account walls.

📦

Collections & Folders

Organize requests into nested collections and folders. Drag, rename, duplicate — just like Postman.

🌐

Environments & Variables

Create environments with {{variables}}. Switch between dev, staging, and production in one click.

🔒

All Auth Types

Bearer, Basic, JWT, OAuth 2.0, AWS Signature, Hawk, NTLM, Digest, API Key, EdgeGrid, ASAP — they're all here.

🎨

Syntax Highlighting

Beautiful JSON/XML syntax coloring with line numbers in both request body and response view.

📋

Pre-request & Test Scripts

Write JavaScript scripts that run before or after requests. Assert response data with pm.test() and pm.expect().

📄

Code Generation

Generate code snippets in JavaScript (Fetch/Axios), Python, Go, PHP, and cURL from any request.

🔁

Import & Export

Import Postman collections (v2/v2.1) with full folder support. Export back to Postman JSON format.

🌈

Light & Dark Mode

Gorgeous light and dark themes that persist across sessions. Easy toggle right from the titlebar.

⚡

Lightweight & Fast

No bloat, no Chromium-heavy frameworks. Starts instantly, uses minimal memory, feels native on macOS.
